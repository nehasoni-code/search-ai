================================================================
SETUP INSTRUCTIONS
================================================================

1. Go to: https://supabase.com/dashboard/project/YOUR_PROJECT_ID/settings/functions

2. Scroll down to the "Secrets" section

3. Click "Add new secret" for each of the 6 secrets above

4. Copy the exact Secret Name and paste your Secret Value

5. Click "Save" after adding each secret

6. Once all secrets are added, the edge functions will automatically have access to them

7. Test your application by creating a new conversation and asking a question

================================================================
TROUBLESHOOTING
================================================================

If you get errors about missing configuration:
→ Double-check that secret names are spelled exactly as shown above
→ Verify your Azure credentials are correct
→ Make sure you've added all 6 secrets

If search returns no results:
→ Verify your Azure Search index contains documents
→ Check that the index name matches exactly
→ Test your search service directly in Azure Portal first
